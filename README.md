# 🌟 LifeLens – AI-Powered Personal Growth & Wellness Companion

## 📖 Overview

LifeLens is an AI-powered wellness and self-improvement platform designed to help users achieve a healthier, more productive, and balanced lifestyle. By combining mood tracking, habit management, goal monitoring, and AI-driven recommendations, LifeLens provides personalized insights that support continuous personal growth.

---

## 🎯 Problem Statement

Existing wellness applications focus on individual areas such as fitness, meditation, or productivity, but fail to provide a holistic view of a user's overall well-being. As a result, users struggle to understand the connection between their habits, emotions, productivity, and personal growth.

---

## 💡 Solution

LifeLens integrates wellness tracking, productivity analysis, habit formation, and goal management into a single intelligent platform. Using Artificial Intelligence, the system analyzes user behavior and delivers personalized recommendations to improve mental well-being, productivity, and lifestyle habits.

## 🚀 Workflow

1. User creates an account.
2. User records moods, habits, goals, and daily activities.
3. Data is securely stored in the database.
4. AI analyzes behavioral patterns.
5. Personalized recommendations are generated.
6. Dashboard displays progress and wellness insights.
7. Users improve their lifestyle through actionable guidance.
